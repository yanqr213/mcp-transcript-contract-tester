# Changelog

## 0.2.0 - 2026-06-08

- Added SARIF 2.1.0 report output for GitHub Code Scanning.
- Added SARIF rule metadata, transcript file locations, line hints, and stable issue fingerprints.
- Added CLI and CI smoke coverage for `--format sarif`.
- Added public GitHub project URLs to package metadata.

## 0.1.0 - 2026-06-08

- Initial local project scaffold.
- Added JSON/JSONL transcript parsing.
- Added MCP-like and custom tool schema snapshot parsing.
- Added JSON-RPC-like pairing, duplicate id, orphan result, tool, argument schema, error format, latency, cost, and baseline diff checks.
- Added Markdown, JSON, and JUnit report formats.
- Added examples, unit tests, and GitHub Actions CI.
